## Abilities Reference

### Passive Abilities

| Passive | Effect |
|---------|--------|
| Regeneration | Heal 2% max HP/sec when not taking damage for 3s |
| Thorns | Reflect 15% of melee damage back to attacker |
| Evasion | 20% chance to dodge attacks |
| Armored | +10 physical defense |
| Magic Resist | 25% reduced magic damage |
| Fire Aura | Deal 5 fire damage to nearby enemies every 1s |
| Flying | Can move over hazards |
| Ethereal | Periodically immune to physical damage (2s immune, 5s vulnerable) |
| Undying | 20% chance to survive lethal damage at 1 HP (30s cooldown) |
| Ambusher | +50% damage on first attack |
| Poison Touch | Basic attacks apply poison (2 dmg/s for 3s) |
| Vampiric | Heal 15% of damage dealt |

### Behavior Patterns

| Behavior | Description |
|----------|-------------|
| Aggressive | Constantly pursues and attacks, no retreat |
| Defensive | Maintains distance, uses ranged attacks, retreats at range 4 |
| Hit and Run | Attacks then retreats for 2 seconds |
| Kiter | Keeps distance, flees when close (3 tiles) |
| Guardian | Protects a location (5 tile radius) |
| Swarm | Weak alone, stronger with allies nearby |
| Ambusher | Waits in hiding, strikes suddenly with bonus damage |

### Triggered Mechanics

| Mechanic | Trigger | Effect |
|----------|---------|--------|
| Enrage | 30% HP | +50% damage, +30% speed |
| Death Explosion | Death | 25 damage in 3 tile radius |
| Split | Death | Spawns 2 smaller versions at 30% HP |
| Teleport When Hurt | On damage | 25% chance to teleport 4-8 tiles away |
| Call for Help | Combat start | Alert allies within 8 tiles |
